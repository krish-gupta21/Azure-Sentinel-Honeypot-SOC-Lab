# Log Analytics Workspace

## Objective

Create an Azure Log Analytics Workspace to centrally collect, store, and query security logs generated by the virtual machine. This workspace will later be connected to Microsoft Sentinel for security monitoring and threat analysis.

---

## Workspace Configuration

| Setting | Configuration |
|---------|---------------|
| Workspace Name | Win11-LogAnalyzer |
| Resource Group | Win11_group |
| Subscription | Azure for Students |
| Region | East Asia |
| Pricing Tier | Pay-as-you-go (Per GB 2018) |

---

## Deployment

A Log Analytics Workspace named **Win11-LogAnalyzer** was created within the same resource group as the virtual machine. The workspace acts as the central repository for collecting Windows Security Events and other telemetry from Azure resources. It will later be integrated with Microsoft Sentinel to enable log analysis, threat hunting, and incident investigation.
